WORK, FOR THE NIGHT IS COMING

1. Work, for the night is
com-ing ;
Work through the morn-ing
hours ;
Work while the dew is
spar-kling ;
Work ‘mid spring-ing flowers.

Work while the day grows
bright-er,
Un-der the glow-ing sun ;
Work, for the night is com-ing,
When man’s work is done.

2. Work, for the night is
com-ing ;
Work through the sun-ny
noon ;
Fill bright-est hours with
la-bor ;
Rest comes sure and soon ;

Give ev-ery fly-ing min-ute
Some-thing to keep in store ;
Work, for the night is com-ing,
When man works no more.

3. Work, for the night is
com-ing ;
Un-der the sun-set skies,
While their bright tints are
glow-ing,
Work, for day-light flies ;

Work till the last beam
fad-eth,
Fad-eth to shine no more ;
Work while the night is
dark-ening,
When man’s work is o’er.
